## How it works

`index.html` is a self-contained landing page generated by [`build.py`](build.py) (stdlib only — no dependencies). It copies the source documents into `papers/` with clean slugs (canonical + prior editions) and lays them out by volume. To rebuild after adding or revising a paper, edit the `VOLUMES` map in `build.py` and run:

```bash
python build.py
```
